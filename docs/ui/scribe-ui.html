Design tokens（镜像，权威来源为 docs/ui/tokens.md）

实现原则：凡标注系统语义色/系统材质的 token，SwiftUI 直接用系统 API，深浅模式自动跟随，不写死颜色值； 只有「自定义」列的 token 需要在 Asset Catalog 中提供 Any/Dark 双值。窗口不设 appearance 覆盖，跟随系统。

token浅色值（聚焦取样）深色值SwiftUI 还原类别
panel.backgroundrgba(252,251,250,0.82)+blur38 sat1.8rgba(30,30,34,0.80)+blur38NSVisualEffectView(.popover/.fullScreenUI, .behindWindow)，以真机聚焦观感为准系统材质（自动）
row.selectedblack @7.5%white @9%Asset「RowSelected」双值；不用强调色反白自定义（聚焦同款）

keycap（按键帽）black @6%（无阴影）white @12%Asset「Keycap」双值自定义（弃纯白）
