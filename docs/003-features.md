# 003 — 功能清单与 Todolist

状态：已采纳（2026-07-17）

## 背景

在 002 架构基础上确定第一版产品的功能范围。用户提出：复制/剪切即写入、快捷键唤起选择、按日期分组、常用时间筛选（今天/昨天/本周/上周/本月）、图片处理、设置页（保留天数、呼出快捷键）。本文档补全其余必要功能并排出实施顺序。

## 功能设计

### 1. 采集（复制/剪切即写入）
- 轮询 `NSPasteboard.changeCount`（200ms），变化即抓取。macOS 层面复制和剪切写剪贴板的行为相同，无需区分。
- 支持类型：纯文本（保留 RTF 原文用于保真回填）、图片、文件引用（URL 单独识别标记，便于展示）。
- **去重置顶**：内容哈希与已有条目相同 → 不新增，刷新该条 `last_used_at` 置顶。
- **隐私处理**：`TransientType`（临时数据，如剪贴板工具自身写入）始终跳过；`ConcealedType`（密码管理器等标记的敏感内容）**默认记录**，条目打上 `is_concealed` 标记，列表中打码显示（●●●●），粘贴/复制功能正常——设置页提供「记录密码类内容」开关，关闭后不再记录。另支持按应用 Bundle ID 排除。
- 记录来源应用（bundle id + 图标），条目上展示。

### 2. 面板（快捷键唤起 + 浏览选择）
- 全局快捷键唤起（默认 ⌘⇧V），非激活 NSPanel，不抢原应用焦点。
- **按日期分组展示**：分组头为「今天 / 昨天 / 7月15日 星期三 / …」，组内时间倒序；Pin 的条目固定在最顶部「置顶」组。
- **时间筛选**：搜索框下一排 chips：全部 · 今天 · 昨天 · 本周 · 上周 · 本月。
- **搜索**：面板唤起即聚焦搜索框，直接打字过滤（文本走 FTS5；图片/文件按文件名和来源应用匹配）。
- 键盘优先：↑↓ 选择、回车回填、⌥回车仅复制、⇧回车以纯文本粘贴、Esc 关闭、⌘1-9 快选前 9 条、⌫ 删除选中条目。
- 条目操作（右键菜单）：粘贴、仅复制、Pin/取消 Pin、删除。
- 长文本条目显示前 3 行预览，选中后侧栏/展开看全文。

### 3. 图片处理
- 原图统一转 PNG 落盘 `Application Support/Scribe/images/<hash>.png`，内容哈希命名天然去重；DB 只存路径、宽高、字节数。
- 生成 ≤200px 缩略图 `<hash>.thumb.png`；**列表只加载缩略图且懒加载**（内存红线关键）。
- 条目显示缩略图 + `1920×1080 · 2.3MB` 标注；空格键看原图预览（Quick Look 风格）。
- 选中回填原图到剪贴板。
- 超过 50MB 的图片跳过不记录（日志提示一次）。
- 条目被淘汰/删除时同步删除对应文件；启动时清理孤儿文件。

### 4. 回填粘贴
- 选中 → 按原类型写回剪贴板 → 关面板 → CGEvent 模拟 ⌘V 粘贴到原前台应用。
- 需 Accessibility 权限：首次使用引导授权；未授权降级为仅复制并提示。

### 5. 设置页
- 呼出快捷键（可录制修改）。
- **保留最近 N 天**（默认 30 天，Pin 条目不淘汰），另设最大条数上限兜底（默认 1000 条，防单日爆量）。
- **记录密码类内容开关**（默认开：记录但列表打码显示；关闭后 `ConcealedType` 内容不记录）。
- 排除应用名单（不记录这些应用的复制）。
- 开机自启开关。
- 存储占用展示 + 「清空全部历史」按钮（需二次确认）。

### 6. 菜单栏
- 菜单栏图标：打开面板、设置、退出。`LSUIElement`，无 Dock 图标。

## 明确不做（第一版）

- iCloud/多设备同步
- 富文本条目的样式化展示（存 RTF 但列表只显示纯文本预览）
- 剪贴板内容编辑
- 历史导出

## 关键取舍

- 保留策略用「天数为主 + 条数兜底」：用户心智是"最近一个月的东西还在"，纯条数上限在高频复制场景会把昨天的内容挤掉。
- 剪切不做特殊标记：NSPasteboard 无法区分复制与剪切，也无此必要。
